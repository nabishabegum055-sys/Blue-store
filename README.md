# 🛒 Blue E-Commerce Web Application

A modern mini **E-Commerce Web Application** built using **Python Flask, HTML, and CSS**.
This project demonstrates a simple online store where users can browse products, add items to cart, place orders, and an admin can view all orders.

---

## 🚀 Features

* 🔐 **User Login & Admin Login**
* 🛍 **Product Store Page**
* 🖼 **Product Cards with Images**
* ➕ **Add to Cart Functionality**
* 🧾 **Cart Page to View Selected Items**
* ✅ **Order Placement System**
* 📦 **Order Confirmation Page**
* 👨‍💻 **Admin Dashboard**
* 📊 **Admin Order List with Total Revenue**

---

## 🛠 Technologies Used

* **Python**
* **Flask**
* **HTML5**
* **CSS3**
* **Jinja2 Template Engine**
